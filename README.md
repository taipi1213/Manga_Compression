# 漫画圧縮ツール (Manga Compression Tool)

ZIPおよびRARアーカイブ内の画像を自動圧縮するGUIツールです。漫画や電子書籍のファイルサイズを効率的に削減できます。

## 主な機能

### manga_asyuku_v2.py（メインツール）

- **ZIP/RAR内画像の一括圧縮** - Caesium CLTを使用した高品質な画像圧縮
- **フォルダ構造の維持** - 複数フォルダ構造のアーカイブも元の構造を保持
- **ドラッグ&ドロップ対応** - ファイルを直接ウィンドウにドロップして処理
- **並列処理** - マルチスレッドで高速処理
- **圧縮後サイズが大きい場合はスキップ** - 元ファイルを自動的に使用
- **エラーログのCSV出力** - 処理履歴をエクスポート可能

### archive_checker.py（破損チェックツール）

- **ZIPファイルの整合性検証** - `zipfile.testzip()`によるチェック
- **画像の開封テスト** - サンプル画像が正常に開けるか確認
- **結果サマリー表示** - 正常/エラー/警告の件数を一目で確認
- **CSV出力** - チェック結果をエクスポート可能

## 動作要件

- Python 3.8以上
- [Caesium CLT](https://saerasoft.com/caesium/) - 画像圧縮エンジン
- 7-Zip（RAR対応用）

### 必要なPythonライブラリ

```bash
pip install pillow tkinterdnd2 send2trash
```

## 使い方

### 画像圧縮

```bash
python manga_asyuku_v2.py
```

1. 起動後、ZIPまたはRARファイルをウィンドウにドラッグ&ドロップ
2. 出力先フォルダを指定
3. 「実行」ボタンをクリック

### 破損チェック

```bash
python archive_checker.py
```

1. 「参照」でチェック対象フォルダを選択
2. 必要に応じて「サブフォルダも対象」にチェック
3. 「チェック開始」をクリック

## 設定項目

### 圧縮設定

| 項目 | 説明 | デフォルト |
|------|------|----------|
| JPEG品質 | 0-100の圧縮品質 | 80 |
| PNG圧縮レベル | 0-6の圧縮レベル | 3 |
| WebP品質 | 0-100の圧縮品質 | 80 |
| 圧縮後サイズが大きい場合スキップ | 元ファイルを使用 | ON |

### 出力設定

| 項目 | 説明 | デフォルト |
|------|------|----------|
| 元ファイル削除 | 処理後に元ファイルを削除 | OFF |
| ファイルサフィックス | 出力ファイル名に追加する文字列 | なし |
